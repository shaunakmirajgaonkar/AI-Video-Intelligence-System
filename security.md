# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.0.x | ✅ Yes |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub Issue.

Instead, contact directly:
- **GitHub:** [@shaunakmirajgaonkar](https://github.com/shaunakmirajgaonkar)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours.

## Notes

- This system runs **100% locally** — no data is sent to external servers
- Ollama inference happens on `localhost:11434` only
- No user data, video frames, or reports are transmitted over the network
